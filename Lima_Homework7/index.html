CNIT 133 Homework 7 - Object Model and Miscellaneous jQuery

Part1:

Create a webpage in which users are allowed to select a page layout and formatting through the use of the className property. Choose at least three form elements from among radio buttons, select items, checkboxes, and buttons to allow the user to change the background color, font size, font family, font style, and font weight. You may use JavaScript or jQuery for this.

Part2:

Write a script that displays a top navigation bar containing a link to your Homework 1 and five other links that will be to the cover page for each of your Homework2 - Homework6. When you mouseover or click on any of these other links then display a drop-down menu that contains the links of their homework parts (part 1, part 2, etc.). If applicable then also include links to any extra credit pages that you have done. Make sure that all links are working. You can use strictly JavaScript or else jQuery - using CSS only will not be acceptable.
